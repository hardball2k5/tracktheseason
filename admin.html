First-time setup — 3 steps
1
Create a Google Sheet. Make a new sheet at sheets.google.com and set up row 1 as headers exactly like this:
team_id team_name date type icon label description Example row: 143 Philadelphia Phillies 2026-04-28 coaching 🎯 Mattingly takes over Rob Thomson fired after 9–19 start. Don Mattingly named interim manager.
2
Publish to web as CSV. In Google Sheets: File → Share → Publish to web → choose Sheet1 → choose CSV → click Publish → copy the URL.
3
Paste the URL in two places. Above in the "Sheet connection" box (to use this admin panel), and also into tracker.html in the SHEET_CSV_URL constant at the top of the script.
4
Use this admin page to generate rows. Fill in the form below → copy the generated row → paste it into your Google Sheet. The site updates within a few minutes.
